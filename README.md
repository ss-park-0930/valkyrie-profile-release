# 발키리 프로파일 한글 패치 (Valkyrie Profile Korean Patch)

PS1 《발키리 프로파일》(일본판, 1999) **비공식 한글 패치 배포** 저장소입니다.
중국어 한화팀 **agemo**의 패치를 토대로 텍스트·메뉴·시스템을 한국어로 옮겼습니다.

> 🌐 패치 사이트: <https://vp.soaptree.dev>

---

## 📥 다운로드

[**Releases**](../../releases) 페이지에서 최신 버전을 받으세요.
각 릴리스에 `valkyrie-profile-korean-patch-vX.Y.Z.zip` 이 있고, 안에 **Disc1·Disc2 PPF**가 들어 있습니다.

---

## 🎮 적용 방법

1. **일본판 원본** 디스크 이미지를 준비합니다 (`.bin` + `.cue`).
   - **Disc1** CRC32 `6768AE3A`
   - **Disc2** CRC32 `9140B9DB`
2. 받은 zip을 풀면 두 개의 PPF가 있습니다.
   - `...-disk1-vX.Y.Z.ppf`
   - `...-disk2-vX.Y.Z.ppf`
3. **PPF 패처**(예: PPF‑O‑Matic) 또는 PPF를 지원하는 에뮬레이터로 각 디스크에 해당 PPF를 적용합니다.
   - 권장 에뮬레이터: **DuckStation**
4. (선택) zip 안의 `SHA256SUMS.txt` 로 파일 무결성을 확인할 수 있습니다.

> 💡 보통 **Disc1**으로 플레이하다가 게임 진행 중 안내에 따라 Disc2로 교체합니다.
> 두 디스크 모두 한글화돼 있습니다.

## ⚠️ 면책 (Disclaimer)

- 이 패치는 **팬이 비영리로** 제작한 **비공식 번역**입니다.
- 적용하려면 **정품 원본 일본판 디스크**가 필요합니다. **ROM/디스크 이미지는 배포하지 않습니다.**
- **개인 학습·비상업적 목적**으로만 사용할 수 있습니다.
- 게임의 모든 권리는 원 제작·발행사(**tri‑Ace / Production I.G. / ENIX 1999**)에 있습니다.
- 본 패치를 상업적 용도로 사용할 수 없습니다.
