# 발키리 프로파일 한글 패치 (Valkyrie Profile Korean Patch)

PS1 《발키리 프로파일》(일본판, 1999) **비공식 한글 패치 배포** 저장소입니다.
중국어 한화팀 **agemo**의 패치를 토대로 텍스트·메뉴·시스템을 한국어로 옮겼습니다.

> 🌐 패치 사이트: <https://vp.soaptree.dev>

---

## 📥 다운로드

[**Releases**](../../releases) 페이지에서 최신 버전을 받으세요.
각 릴리스에 `valkyrie-profile-korean-patch-vX.Y.Z.zip` 이 있고, 안에 **Disc1·Disc2 PPF**가 들어 있습니다.

## 🎮 적용 방법

**준비물**: 일본판 원본 디스크 이미지(`.bin` + `.cue`) — **Disc1** CRC32 `6768AE3A`, **Disc2** CRC32 `9140B9DB`. zip을 풀면 `...-disk1-vX.Y.Z.ppf` / `...-disk2-vX.Y.Z.ppf` 두 PPF가 있습니다. **항상 원본에 적용**하세요(이미 패치된 파일에 덧씌우지 마세요).

### ⭐ 방법 A — DuckStation 외장 적용 (원본 미수정, 가장 간편)
PPF를 디스크 이미지와 **같은 폴더·같은 이름**으로 두면 DuckStation이 실행 시 자동으로 얹어줍니다. BIN을 굽지 않아 가장 쉽고, 버전 교체 시 **PPF만 바꾸면** 됩니다.

1. PPF 이름을 디스크 이미지와 똑같이 맞춥니다(확장자만 `.ppf`).
   - 예: `Valkyrie Profile (Disc 1).bin` → `Valkyrie Profile (Disc 1).ppf`
   - disk1 패치는 Disc 1, disk2 패치는 Disc 2 이름으로 각각.
2. **설정 → 콘솔 → CD-ROM** 의 **이미지 패치 적용**(Apply Image Patches)을 체크합니다.
3. 게임을 실행하면 자동 적용됩니다(원본 `.bin` 그대로 유지).

> DuckStation 등 일부 에뮬레이터에서만 지원합니다(현재 PPF). 실기·다른 에뮬레이터는 방법 B·C로 BIN에 직접 적용하세요.

### 방법 B — PPF 패처(GUI)로 BIN에 직접 적용
**PPF‑O‑Matic** 같은 패처로 원본 `.bin`에 해당 PPF를 적용해 패치된 이미지를 만듭니다.
- ⚠ 반드시 **깨끗한 원본 사본**에 적용하세요(이미 패치된 파일에 덧씌우면 안 됩니다).

### 방법 C — 명령줄 패처
`applyppf3 a "원본.bin" "패치.ppf"` 형태로 적용합니다(B와 동일하게 원본 기준).

✅ (선택) zip 안의 `SHA256SUMS.txt`로 파일 무결성을 확인할 수 있습니다.

> 💡 보통 **Disc1**으로 플레이하다가 게임 진행 중 안내에 따라 Disc2로 교체합니다. 두 디스크 모두 한글화돼 있습니다.

## 🛠 에뮬레이터 안내 / 문제 해결

- **권장 에뮬레이터**: **DuckStation** — 컨트롤러·디스크 교체를 자동 처리해 가장 간편합니다.
- **컨트롤러가 안 먹힐 때**: 에뮬레이터의 컨트롤러 설정을 확인하세요.
  - ePSXe는 **F5**로 아날로그/디지털 모드를 전환합니다.
- **CD2(Disc2) 교체**: 게임 진행 중 Disc2로 넘어갈 때,
  - 패치된 Disc2 이미지로 바꾸거나, 에뮬레이터의 **디스크 교체** 기능을 사용하세요.
  - 외장 PPF 방식이 교체 후 안 먹히면 **`.bin`(ISO)에 PPF를 직접 적용**한 이미지를 쓰세요.
- **패치 업데이트**: 새 버전은 **항상 일본판 원본**에 다시 적용하세요. 업데이트 전 원본 백업을 권장합니다.

> 게임 내 부팅 화면의 안내(구버전 에뮬레이터·플러그인 설정 등)는 대부분 옛 환경용입니다.
> 최신 환경에서는 **DuckStation** 사용을 권장합니다.

## ⚠️ 면책 (Disclaimer)

- 이 패치는 **팬이 비영리로** 제작한 **비공식 번역**입니다.
- 적용하려면 **정품 원본 일본판 디스크**가 필요합니다. **ROM/디스크 이미지는 배포하지 않습니다.**
- **개인 학습·비상업적 목적**으로만 사용할 수 있습니다.
- 게임의 모든 권리는 원 제작·발행사(**tri‑Ace / Production I.G. / ENIX 1999**)에 있습니다.
- 본 패치를 상업적 용도로 사용할 수 없습니다.
